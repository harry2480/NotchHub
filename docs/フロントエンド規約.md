## Server Components / Client Components の使い分け

- **デフォルトは {{DEFAULT_COMPONENT_TYPE / 例：Server Component}}。** `{{CLIENT_DIRECTIVE / 例：'use client'}}` は必要な場合のみ付与
- Client Component が必要なケース: ユーザーインタラクション（フォーム操作、ボタン制御等）、リアルタイム更新、Web API依存（window等）の使用
