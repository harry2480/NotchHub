# NotchHub - サービスコンセプト

## サービス概要

NotchHub は、Mac のノッチ（およびノッチなし Mac では擬似ノッチ）を「共有」と「一時保管」のハブとして活用する常駐型 macOS ネイティブアプリである。ファイルやテキストをノッチへドラッグするだけで AirDrop・Share・Shelf へ振り分けられ、日常のファイル受け渡しと一時保管を最短動線で完結させる。

本プロダクトは **「AI アシスタント管理アプリ」ではなく、「ノッチを活用した共有・一時保管ハブ」** として定義する。

```text
主役: AirDrop + Shelf
補助: AI CLI Monitor / Calendar / Media
```

---

## コンセプト

### 1. 共有を最短動線にする（AirDrop / Share）

**目的**: ファイル共有の「掴む → 送る」を、ノッチへのドラッグ 1 アクションで完結させる。

**機能**:
- ドラッグ中に画面上部中央へ近づくとノッチが展開し、3 Zone（Shelf / Share / AirDrop）を提示
- 右 Zone への Drop で AirDrop 共有 UI を即起動
- 中央 Zone への Drop で macOS Share Sheet を起動（Mail / Messages / Notes 等、Discord・LINE も Share Sheet 経由）

### 2. 一時保管を気軽にする（Shelf）

**目的**: 「あとで使う・あとで送る・一時保管する」を、コピーを作らず参照保持で実現する。

**特徴**:
- ファイル / フォルダ / テキスト / URL / Markdown / 画像 / スクリーンショットに対応
- ファイルは BookmarkData による参照保持（コピーしない）。元ファイル削除時は Shelf からも自動削除
- テキスト類は SQLite に保存し、共有時のみ一時ファイル（.txt / .md / .webloc）を生成
- スクリーンショット（⌘+Shift+3/4/5）を自動検出して Shelf へ追加
- 全文検索（ファイル名・フォルダ名・URL・テキスト/Markdown 本文）を MVP から提供

### 3. 邪魔をしない常駐 UI（Notch）

**目的**: 通常時はノッチに同化し、必要なときだけ展開する。

**特徴**:
- 通常時は何も表示せずクリック可能領域としてのみ存在
- 展開は「ドラッグ接近」「クリック」「AI 承認待ち」の明示的トリガーのみ（Hover・接近では発火しない）
- 最小状態は優先度順（AI 承認待ち > ドラッグ中 > AirDrop/Share 実行中 > メディア再生中 > 予定 10 分前）で 1 状態のみ提示

### 4. 補助情報をノッチに集約（AI Monitor / Calendar / Media）

**目的**: 開発・予定・再生中メディアを、コンテキストスイッチなくノッチから把握・操作する。

**主要機能**:
- **AI CLI Monitor**: Claude Code / OpenAI Codex CLI / Antigravity CLI のセッション・状態・承認待ちを表示。承認待ち発生時はノッチを自動展開し Approve / Deny
- **Calendar**: EventKit 経由で Calendar.app 準拠の Next / Today を表示（Google Calendar は同期済みのものを表示）
- **Media**: Apple Music / Spotify の曲名・アーティスト・アートワーク表示と Play/Pause・Next・Previous 操作

---

## 期待される効果

1. **共有コストの最小化** - 既存の Finder → 右クリック → 共有メニューの多段操作を、ノッチへのドラッグ 1 操作に短縮
2. **一時保管の習慣化** - コピーを作らない参照保持により、ストレージを汚さず気軽に「あとで」を溜められる
3. **コンテキスト維持** - AI 承認待ち・予定・メディアをノッチに集約し、アプリ切り替えなしで把握・操作
4. **日常の非侵襲性** - 通常時はノッチに同化し、明示操作時のみ展開するため常駐していても邪魔にならない

## 技術スタック（概要）

- 言語: Swift
- UI: SwiftUI / AppKit
- 永続化: SQLite
- 主要 API: NSSharingService / NSSharingServicePicker / EventKit / Accessibility API / Security Scoped Bookmark / File System Events / Launch Services / Local Socket
