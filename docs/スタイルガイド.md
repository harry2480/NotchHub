## {{STYLE_RULE_TITLE_1 / 例：ビューポート高さ: `dvh` を使う}}

### ルール

- {{RULE_PROHIBITED_1 / 例：`h-screen` / `min-h-screen` は**使用禁止**}}
- {{RULE_ALLOWED_1 / 例：代わりに `h-dvh` / `min-h-dvh` を使用する}}
- {{RULE_EXTRA_1 / 例：`calc()` 内の `100vh` も `100dvh` に置き換える}}

### 理由

{{RULE_REASON_1 / 例：モバイルブラウザ（iOS Safari, Android Chrome 等）では、アドレスバーやナビゲーションバーの表示/非表示により、`100vh` が実際の可視領域と一致しない。`dvh`（Dynamic Viewport Height）を使うことで、ブラウザ UI の状態に応じた正確な高さを取得できるため。}}

### {{COMPARISON_TABLE_TITLE / 例：ビューポート単位の比較}}

| {{TABLE_HEADER_1 / 例：CSS 単位}} | {{TABLE_HEADER_2 / 例：Tailwind クラス}} | {{TABLE_HEADER_3 / 例：説明}} |
|----------|----------------|------|
| `100vh` | `h-screen` | ブラウザ UI を含む固定サイズ。モバイルで問題になりやすい |
| `100dvh` | `h-dvh` | ブラウザ UI の表示/非表示に応じて動的に変化（推奨） |
| `100svh` | `h-svh` | ブラウザ UI が表示された状態の最小ビューポート |
| `100lvh` | `h-lvh` | ブラウザ UI が非表示の状態の最大ビューポート |

### 具体例

```tsx
// NG
<div className="{{NG_CLASS_EXAMPLE_1 / 例：min-h-screen bg-gray-50}}">
<div className="{{NG_CLASS_EXAMPLE_2 / 例：h-screen md:h-[calc(100vh-96px)]}}">

// OK
<div className="{{OK_CLASS_EXAMPLE_1 / 例：min-h-dvh bg-gray-50}}">
<div className="{{OK_CLASS_EXAMPLE_2 / 例：h-dvh md:h-[calc(100dvh-96px)]}}">
```

### ブラウザサポート / 備考

{{BROWSER_SUPPORT_NOTE / 例：`dvh` は主要ブラウザで広くサポート済み（Chrome 108+, Safari 15.4+, Firefox 101+）。古い環境への対応が必要な場合のみフォールバックを検討すること。}}
