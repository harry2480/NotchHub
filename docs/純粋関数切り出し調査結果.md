## 対象外と判断したもの（例）

- React Componentの純粋なJSXレンダリング部分
- データベース等の外部システム呼び出しに完全に依存する手続き的コード
- `process.env` に依存する実行環境設定
- DOM API依存（Window, documentへの直接アクセス）
- 定数定義（メタデータ、マッピング定義など実行ロジックを持たないもの）
- 既にテスト済みのファイル
